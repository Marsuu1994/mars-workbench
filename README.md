# Mars Workbench

A kanban-first personal productivity app: a weekly period planner built around a drag-and-drop kanban board, an Eisenhower priority matrix, and AI-assisted plan creation. Next.js + Supabase + Prisma, deployed on Vercel.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
